# Echo Builder

## Configuration

- NodeJS >= 22
- Yarn

## Clone development project

1. Clone the [GitHub](https://github.com/humonio-group/echo-builder) project

```shell
git clone https://github.com/humonio-group/echo-builder.git Echo-Builder
```

2. Install dependencies

```shell
cd Echo-Builder
yarn install
```

3. Start the development server

```shell
yarn dev 
```
